# Financial Manager API

## Installation

1. Install dependencies inside "/api":

```cmd
cd api
npm install
```

## Testing

1. Run database docker container:

```cmd
docker-compose up postgres
```

2. Run npm script:

```cmd
npm run test
```

## Development

1. Run database docker container:

```cmd
docker-compose up postgres
```

2. Run npm script:

```cmd
npm run dev
```

## Production

Run all docker containers inside project root folder:

```cmd
docker-compose up
```
